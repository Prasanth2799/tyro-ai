# 🛡️ TyroAI

TyroAI is an AI-powered web application that helps users identify whether a message is **Safe**, **Suspicious**, or a **Potential Scam**. Simply paste any SMS, WhatsApp, Email, or other text message, and TyroAI uses AI to analyze it, explain why it's risky (or safe), and provide actionable recommendations to help users stay protected from phishing, fraud, and social engineering attacks.

## ✨ Features

- 🤖 AI-powered scam and phishing detection
- 📋 Paste and analyze suspicious messages instantly
- 🚨 Detect phishing, fraud, fake banking alerts, and social engineering attempts
- 📊 AI-generated confidence score and risk assessment
- 📝 Detailed explanation of detected threats
- 💡 Personalized safety recommendations
- ⚡ Modern, fast, and responsive user interface

## 🛠️ Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript
- Gemini AI (`@google/genai`)

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Prasanth2799/tyro-ai.git
cd tyro-ai
```

### Install dependencies

```bash
cd client
npm install
```

### Configure environment variables

Create a `.env` file inside the `client` folder:

```env
VITE_GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

### Start the development server

```bash
npm run dev
```
